# Проект "MyService" — Многостраничный сайт с использованием Tailwind CSS

## Описание проекта
Учебный проект, демонстрирующий создание многостраничного сайта с использованием CSS-фреймворка Tailwind CSS. Проект включает:

- Главную страницу с навигацией
- Страницу контактов с формой обратной связи и картой
- Страницу FAQ с аккордеоном
- Креативную страницу 404

## Технологии
- HTML5
- Tailwind CSS (CDN)
- JavaScript (чистый, без фреймворков)
- Google Fonts (Inter)
- Google Maps iframe
